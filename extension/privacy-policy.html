Privacy Policy

GitHub PR Approved Viewer  |  Last updated: March 4, 2026

概要 / Overview

✓ データ収集なし

GitHub PR Approved Viewer は、GitHub の Pull Request ページに CODEOWNERS 承認状況パネルを追加する Chrome 拡張機能です。

本拡張機能は、いかなる個人情報・ユーザーデータも収集・保存・送信しません。

収集するデータ / Data Collection

本拡張機能は以下のデータを 収集しません：

  • 個人を特定できる情報（氏名、メールアドレス等）
  • 認証情報（パスワード、トークン等）
  • 閲覧履歴・行動ログ
  • その他いかなるユーザーデータ

取得した情報（CODEOWNERS の内容、変更ファイル一覧、チームメンバー情報）はすべて ページ表示のみに使用 され、外部サーバーへの送信・ローカルストレージへの保存は行いません。

アクセスする情報 / Data Accessed

本拡張機能は GitHub PR ページを表示する際に以下の情報にアクセスします：

  • 現在表示中の Pull Request のファイル変更一覧（github.com 内のページを fetch）
  • リポジトリの CODEOWNERS ファイルの内容（github.com 内のページを fetch）
  • GitHub チームのメンバー一覧（github.com 内のページを fetch）
  • Pull Request サイドバーのレビュアー承認状態（ページ DOM の読み取り）

すべての通信は https://github.com 内に限定されます。第三者サービスへのデータ送信は一切行いません。

第三者への提供 / Third-Party Sharing

いかなる情報も第三者に販売・提供・共有しません。

リモートコード / Remote Code

本拡張機能は外部の JavaScript ファイルを読み込みません。eval() や動的コード実行も使用しません。すべてのロジックは拡張機能パッケージに含まれています。

変更 / Changes to This Policy

プライバシーポリシーを変更した場合は、このページの「Last updated」日付を更新します。
